greppwd - показывает местонахождение
cd .. на каталог вверх
mkdir "name" - создает директорию
ls -la выводит список каталогов или файлов
mkdir catalog catalog1 catalog2 - создаст три каталога с такими неймами 
touch - создает файл (touch test.txt например)
cat >> (название файла) включает редактирование файла, через cntrl + C закрывает
cat (название файла) чтение файла
cat > (навзения файла) создает файл и переводит в режим редактирования

history выводит историю команд за сессию

vim test.txt - открывает файл в вим
i - режим редактировния
esc - выйти из режима редактирования
:wq выйти из вима
 -
mv каталог/файл.расширение каталог/имя_файла.расширение - перемещение файла
cp - копирование файла, аналогично mv
rm файл.расширение удаляет файл (не папку!)
rmdir удаление каталога
rm калог -r удаление каталога и всех файлов в нём
mv файл.раширение файл.расширение - переименовние файлады 
cd - выход в корень
